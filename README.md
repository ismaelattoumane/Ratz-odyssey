# Ratz-Odyssey
Projet de jeu RPG 2D / 2.5D en pixel-art développé par le Club Informatique. Le joueur incarne un rat parcourant un univers rempli de PNJ pouvant devenir alliés ou ennemis selon les choix effectués. Le jeu proposera plusieurs fins ainsi qu’une “vraie fin” influencée par les décisions du joueur.
